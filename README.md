# HDLBits Verilog Solutions

This repository contains my solutions to the HDLBits problem sets,
implemented in synthesizable Verilog HDL.

## Topics Covered
- Verilog Basics
- Combinational Logic
- Sequential Logic
- Finite State Machines (FSM)
- Arithmetic Circuits
- Parameterized Designs

## Purpose
- Improve RTL design skills
- Practice interview-style problems
- Build a strong digital design portfolio

## Disclaimer
These solutions are for learning and reference purposes.
If you are using HDLBits, please try solving the problems yourself first.
